Open Source

Open source software is code that is designed to be publicly accessible. Prime examples include Linux, Firefox & Android. If you’re a new open source contributor, the process can be intimidating. How do you find the right project? What if you don’t know how to code? What if something goes wrong? That is why we have carefully curated some resources on all you need to know about open source
